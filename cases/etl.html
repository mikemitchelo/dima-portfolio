Задача

Аналитика без кода

Аналитики и дата-инженеры тратили часы на настройку ETL-пайплайнов через SQL и скрипты. Каждое изменение требовало участия разработчика, а бизнес-пользователи были полностью исключены из процесса.

Нужен был инструмент, который позволил бы настраивать сложные потоки данных без технической подготовки — и вписывался в существующую BI-платформу.

Решение

Визуальный конструктор связей

Спроектировал drag-and-drop интерфейс для создания ETL-пайплайнов. Пользователь соединяет источники данных, трансформации и выгрузки визуальными блоками — без единой строки кода.

Разработал систему валидации в реальном времени: ошибки конфигурации видны сразу, до запуска пайплайна. Сложные модели данных стали читаемыми схемами.

Результат

Скорость и доступность

Время настройки типового пайплайна сократилось в 3 раза. Инструментом начали пользоваться аналитики без технического бэкграунда — задачи перестали накапливаться в очереди к разработчикам.
